# Blueprint 003
# User Profile

**Projekt:** Alpine Career
**Version:** 1.0
**Status:** Draft

---

# Ziel

Das Benutzerprofil bildet die zentrale Datenquelle von Alpine Career.
Alle Informationen werden einmal erfasst und für den gesamten Bewerbungsprozess wiederverwendet.

---

# 1. Persönliche Daten

- Vorname
- Nachname
- Geburtsdatum
- Adresse
- PLZ
- Ort
- Land
- Telefonnummer
- E-Mail

---

# 2. Berufliches Profil

- Aktuelle Position
- Berufserfahrung
- Branche
- Verfügbar ab
- Beschäftigungsgrad
- Führerschein
- Reisebereitschaft

---

# 3. Wunschprofil

- Zielberuf
- Wunschpensum
- Wunschlohn
- Regionen
- Homeoffice
- Max. Pendelzeit
- Vertragsart
- No-Gos

---

# 4. Ausbildung

- Schulen
- Lehren
- Studien
- Weiterbildungen
- Zertifikate

---

# 5. Kompetenzen

- Fachkompetenzen
- Soft Skills
- Sprachen
- IT-Kenntnisse
- Führungsverantwortung

---

# 6. Dokumente

- Lebenslauf
- Arbeitszeugnisse
- Diplome
- Zertifikate
- Portfolio
- Motivationsschreiben

# 7. Integrationen

# 8. Bewerbungsziele

- Monatliches Bewerbungsziel (z. B. 12)

- Automatische Stellenvorschläge

---

# 9. KI-Profil

Die KI merkt sich:

- bevorzugte Firmen
- bevorzugte Branchen
- bevorzugte Stellen
- bisherige Bewerbungen
- Interviewerfahrungen
- häufige Absagen
- erfolgreiche Strategien

---

# 10. Datenschutz

Der Nutzer kann jederzeit:

- Daten exportieren
- Daten löschen
- Integrationen trennen
- AI Memory zurücksetzen

---

# Offene Punkte

Diese Informationen dienen später als Grundlage für:
- Datenbankmodell
- Onboarding
