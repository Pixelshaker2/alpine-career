# Product Rules

1. Keine Bewerbung ohne Freigabe.
2. Der Nutzer gibt Daten nur einmal ein.
3. Jede Bewerbung erhält einen eigenen Workspace.
4. Google Workspace ist das primäre Ökosystem.
5. Telegram ist das Hauptinterface.
6. Das Dashboard dient nur für Einrichtung, Kontrolle und Historie.
7. Jede Funktion muss Zeit sparen oder die Erfolgswahrscheinlichkeit erhöhen.
